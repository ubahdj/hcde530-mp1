# Mini Project 1 — Seattle Coffee Shop Licenses

HCDE 530 analysis of Seattle business license data (coffee-related trade names).

## Contents

- **`Mp1.ipynb`** — full analysis notebook (overview, data profile, Plotly charts, conclusions)
- **`mp1.md`** — competency notes and findings summary
- **`figures/`** — static PNG exports of the three charts (kaleido)

## Data source

[Seattle Open Data — Business licenses](https://data.seattle.gov/resource/wnbq-64tb.json)

## Setup

```bash
pip install jupyter plotly kaleido pandas
jupyter notebook Mp1.ipynb
```

Run the pip install cell in the notebook first, then **Run All**.
